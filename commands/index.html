Bot Commands

Welcome to the help page for all of the Echoes bots. This has a list of all of _saltshaker's commands.

Teleport Commands
  • !hub This allows you to teleport to The Hub. The Hub is located in a custom dimension added via a data-pack, and is intended for trading with other players. At the Hub, you can find multiple villagers to trade with, should you need materials for your base or your skyblock island (you can spawn an island by typing !skyblock).
  • !sethome name This functions like /sethome on Spigot servers. You may have up to 11 homes. To set a home, type !sethome followed by the name of the home (example: !sethome mineshaft sets a home called "mineshaft"). Teleport to a home by typing !home and specifying its name (example: !home mineshaft teleports you to a home called "mineshaft," if one is set). Additional home commands include; !delhome (deletes a home, but you must specify the name of the home you'd like to delete), !listhomes (lists all homes and their coordinates), !undo (useful if you accidentally overwrite a home), and !color homes (Changes the color of the home text in chat). You must be at full health in order to teleport to a home (this is to prevent tp'ing away in combat). Thanks to djie for writing the homes data pack for Echoes. You can check out his Github page by clicking here.
  • !spaceship Spawns in a space ship for you, in the space dimension. You must farm certain materials in order to spawn a space ship (lots of quartz, obsidian, a respawn anchor, lightning rod, etc. Type !spaceship or !skyblock in-game to see a full list of materials). After successfully executing this command, the player "blasts off" into space, complete with flame & smoke particles, a speedy levitation effect, and explosion sounds. Each space ship is piloted by an axolotl and comes equipped with basic survival materials.
  • !back Teleports you to your previous location. Yes, it saves your death location. There is a 1-minute cooldown on this command.
  • !tpa username Teleports you to the player you specify. They must click the message that appears to them in chat, or type !accept your-username to accept your request.
Player-specific Commands
These commands allow the players to view information about themselves or to escape from bed traps.
  • !unbedtrap OR !escape The !unbedtrap command is useful for escaping a bed trap. It moves your spawnpoint to (0, 0).
  • !clearmyinventory This will clear your inventory. After you execute this command, you cannot get your items back.
  • !minecartafk OR !afk Grays out your name in chat and in the tab list and gives you the [AFK] prefix. !minecartafk will delete oncoming Minecarts between 20 and 25 blocks away. Useful if going AFK while in a minecart. Try typing !afkpussy, !afksleeping, or !afkshitting. To remove, type !unafk or !unminecartafk.
  • !suicide method

  • !radio on OR !radio off This command turns the server radio on/off for you. Make sure you have server resource packs enabled and that your sound is on, or else this won't work. It may take up to 6 minutes for music to start playing. To skip a song, type !skip. The radio mostly plays 8-bit versions of 80s music (with some Kazakh nationalist music and JustinRPG & Chris Chan songs.). _saltshaker narrates news stories interspersed between the songs, covering recent server events and detailing where you can find various custom NPCs located throughout the map.
Server-wide Commands
These commands allow the player to interact with gamerules, examine the server specs (like the current TPS), or to learn about other players.
  • !advancements on/off Use this command to toggle whether advancements appear in chat. Can be useful for clearing up achievement spam. !advancements on will turn gamerule announceAdvancements on, !advancements off will turn it off. To query the announceAdvancements gamerule, type !advancements.
  • !deathmessages on/off Use this command to toggle whether death messages appear in chat. This can be useful to clear up death spam, like if someone is being spawn killed.!deathmessages on will turn gamerule showDeathMessages on, !deathmessages off will turn it off. To query the showDeathMessages gamerule, type !deathmessages.
  • !lu username Look up a player's past usernames, skins, and the servers they frequent.
  • !bio set message Use this command to set your bio. You can view other players' biographies by typing !bio username.
  • !discord Displays the Discord server invite.
  • !rules Displays the rules.
  • !vote Links to the server's vote pages.
  • !clearlag Removes all items, ender crystals, tnt, and experience orbs within a 32 block radius of the player.
  • !stats This command displays some basic statistics about you. To get a more complete list, type !fullstats
  • !announce message Displays a message in bold red text as an announcement to the server chat. Has a 10 second cooldown.
  • !mount Allows you to ride on the head of any player or entity. You will automatically mount the player/entity you're closest to.
  • !seen username Shows you the date of a players' last login (mm/dd/yy).
  • !lastwords username Displays the last sentence someone posted to chat before logging out.
  • !dox username Displays some made-up information about the specified user.
  • !leaderboard set scoreboard objective

  • !tps Will give the current TPS (rate of ticks per second). This is a good indicator of server lag. The lower the number, the laggier it will be. To get a more accurate picture of the TPS, type !tps average instead. This averages the TPS over the course of 5-6 seconds.
  • !online Lists all users currently logged in. If there are more than 15 users online, _saltshaker will begin to speak.
