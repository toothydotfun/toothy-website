Welcome

Echoes is a Minecraft community for those without community. We are eccentric and unreasonable. We don't like to take ourselves too seriously.

Server IP

The IP address for our vanilla server is always echoes.fun, version 1.18.2. For our modded server, the IP is always modded.echoes.fun (or echoes.fun:25545), version 1.17.1. For the time being, both Echoes servers are only accessible from Minecraft: Java Edition. From time to time, we will set up novelty IP addresses from which one can access Echoes. Current active ones are as follows:
  • obamasex.com AND modded.obamasex.com

  • cumf.art

Non-profit Server

We don't sell ranks, accept donations, or sell any merchandise. All promotional and hosting costs are paid by the server owner's personal budget. If you wish to support the server, all we ask is that you tell your friends to check it out. Maybe even ask them to join our Discord server Or if you'd like some in-game rewards, consider voting for us.

Big Changes

Since this page was last updated, some big changes have taken place on Echoes. They are as follows:
  • 15 new songs were added to Echoes Radio. Turn on server resource packs & type !radio on upon login to begin listening.
  • Several new saltshaker commands like !spaceship were added, as well as some easter egg commands.
  • A rule against hacking was implemented. This may be subject to change.
  • Echoes: Vanilla also underwent a map reset for 1.18. We don't intend for another map reset for several years to come. Our last map was 5 years old. RIP.
  • We've bought several new novelty domains, most notably sexwithkids.com (no, CP is not allowed on Echoes).
  • Gamer2070, Evooper, and DUKE_G00KEM joined our admin team.
  • A three-way chat relay between Echoes: Modded, Echoes: Vanilla, and the Discord server was established.
  • Several dimensions on Echoes: Vanilla (Skyblock, Superflat, Cumworld, Aether, and the Vagina Dimension) will soon be opening to gameplay.
